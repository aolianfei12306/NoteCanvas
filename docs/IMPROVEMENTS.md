# NoteCanvas 后续改进清单

## 编辑体验

- 为文本块增加明确的“选中但不编辑”和“正在编辑”两级状态，避免单击、拖拽、输入之间互相抢焦点。
- 为文本块提供对齐线、吸附、键盘微调和批量选择能力，让自由画布上的排版更可控。
- 将文本块工具栏从块内浮层升级为靠近选区的独立浮动工具条，减少遮挡页面顶部内容的情况。
- 为图层面板增加内联重命名和删除图层功能，替代当前的双击和系统 prompt。

## 数据模型与兼容

- 为 `schemaVersion` 增加真实迁移版本号，避免后续多次结构升级都挤在同一个规范化函数里。
- 为页面、图层、形状、文本块增加更系统的单元测试，覆盖旧数据迁移、隐藏图层导出、锁定图层编辑限制等场景。
- 为 Undo/Redo 增加可配置历史深度和跨笔记清理策略，避免长时间使用时内存持续增长。
- 考虑把设置从 `localStorage` 迁移到 Electron 主进程文件存储，方便备份、恢复和跨渲染进程一致性。

## 画布与绘制

- 为画笔笔迹增加平滑算法和压力/速度模拟，让鼠标和触控板绘制更自然。
- 为几何图形增加选中后编辑、颜色修改、复制、删除、旋转和比例锁定。
- 为 `Ctrl + 滚轮` 缩放补充快捷键缩放、重置缩放、适配宽度和适配整页。
- 为多页视图增加页面缩略导航，长笔记中定位会更快。

## 导出与文件

- 为导出增加整页导出、整笔记导出、多页 PDF 导出和透明背景选项。
- 当前隐藏图层不参与导出，后续可增加“导出时临时包含/排除图层”的开关。
- 为本地工作库增加自动备份、手动恢复和损坏 JSON 的恢复提示。

## 开发质量

- 增加 Playwright 或 Electron 端到端用例，覆盖重命名、关闭窗口、缩放拖拽、图层显隐、导出入口等交互。
- 增加 UI 文案完整性检查，扫描源码中的异常 `??`、乱码和未闭合标签，防止本地化文字回归。
- 为大文件组件拆分子组件，例如页面控件、图层面板、导出工具条、画布页面和绘制层。
- 为 Windows 开发环境固定文件编码检查流程，避免 PowerShell 管道导致中文被替换为问号。
